Usage :

cd src/

--Pour une classification par descripteur de couleurs : --
./classificationColor.sh [chemin du fichier image] --> Ex: ./classificationColor.sh ../test/bird.jpg

--Pour une classification par descripteur de texture : --
./classificationSift.sh [chemin du fichier image] --> Ex: ./classificationSift.sh ../test/bird.jpg

--Pour une classification par descripteur de couleurs et de texture : --
./classificationFusion.sh [chemin du fichier image] --> Ex: ./classificationFusion.sh ../test/bird.jpg
